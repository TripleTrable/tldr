# termdown

> Command-line countdown timer.

- Start a stopwatch:

`termdown`

- Start a 1 minut and 30 second coundown:

`termdown {{1m30s}}`

- Start a 1m 30s countdown with binking terminal at the end:

`termdown {{1m30s}} -b`

- Show title above countdown:

`termdown {{1m30s}} -T "{{Interesting title}}"`

- Show current time:

`termdown -z`
